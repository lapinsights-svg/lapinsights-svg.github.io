# BREXIT : LE BIAIS DES SONDAGES

![Brexit sondages](/images/actualites/brexit-biais.png)

Dans le graphique que l on trouve sur ce site, on voit que les Britanniques regrettent d avoir quitte l UE :

https://www.statista.com/statistics/987347/brexit-opinion-poll/

Je pars du principe que le sondage n est pas entierement truque.  
Mais je remarque que la polarite semble trop parfaite pour etre honnete.

Le biais, c est **l apparente symetrie** du graphique.

![Brexit sondages](/images/actualites/brexit-opinion1.png)

## Que mesurent vraiment ces sondages ?

Les sondages YouGov/Statista indiquent qu environ **56 %** des repondants estiment que le Brexit etait une mauvaise decision.

Mais ce chiffre **n inclut pas les indecis**.

Les chiffres reels sont plutot :

- **48 a 50 %** : mauvaise decision  
- **28 a 32 %** : bonne decision  
- **15 a 23 %** : indecis

Ce dernier groupe est massif et change completement la lecture.

## La symetrie est un artefact statistique

Elle vient de choix methodologiques :

- deux categories exclusives,  
- parts relatives totalisant 100 %,  
- suppression des indecis,  
- lissage statistique.

Cela cree une **illusion de polarisation**.

Si l on reconstruisait le sondage avec les indecis, la symetrie disparaitrait totalement.

![Brexit sondages](/images/actualites/brexit-opinion2.png)

## Le Brexit n est pas la cause principale des difficultes economiques

Les difficultes du Royaume-Uni sont largement partagees par les pays de l UE :

- inflation,  
- croissance faible,  
- choc Covid,  
- dependance energetique.

Les trajectoires France / Royaume-Uni sont quasi identiques.

Le Brexit n a pas cree un effondrement economique, mais il n a pas ameliore la situation non plus.

## Les biais cognitifs influencent l opinion

- biais de contraste  
- biais d attribution  
- biais de cadrage  
- biais de disponibilite

Ces biais expliquent pourquoi les recits simplifies dominent.

La suppression des indecis amplifie :

- l illusion de polarisation,  
- les recits politiques simplistes,  
- la perception publique,  
- les strategies des partis.

## La presentation influence l opinion

Quand un graphique montre :

- **56 %** mauvaise decision  
- **31 %** bonne decision

… les gens pensent :

"Une majorite regrette, donc moi aussi je peux le dire."  
"Je ne suis pas seul."  
"Le pays a change d avis."

La presentation influence la perception, qui influence l opinion, qui influence la politique.

C est un cercle vicieux.

## Conclusion

Les sondages montrent un regret croissant, mais **pas une majorite absolue**.  
La symetrie parfaite du graphique est un artefact.  
Les indecis representent un bloc essentiel.  
L opinion publique est influencee autant par des biais cognitifs que par des faits economiques.

Le Brexit n est ni la cause principale des difficultes britanniques, ni un remede.
